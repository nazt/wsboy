# /recap — เริ่ม Session ใหม่

## มันคืออะไร?

เวลาเปิด Claude ขึ้นมาใหม่ บางที Claude ลืมว่าเมื่อกี้ทำอะไรค้างไว้

`/recap` ช่วย**สรุปให้** ว่า:
- ทำอะไรไปแล้ว
- มีอะไรค้าง
- ควรทำอะไรต่อ

---

## ใช้ยังไง?

### แบบเร็ว (ปกติ)
```
/recap
```
→ สรุปใน 10 วินาที

### แบบละเอียด
```
/recap --rich
```
→ อ่านไฟล์ย้อนหลัง + วิเคราะห์เพิ่ม

---

## ได้อะไรออกมา?
