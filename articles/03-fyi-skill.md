# /fyi — จดโน้ตก่อนลืม

## มันคืออะไร?

`/fyi` = "For Your Information"

เหมือน**กระดาษโน้ตเหนียว** ที่ Claude ช่วยเก็บให้

---

## ใช้ยังไง?

### แบบปกติ
```
/fyi ต้องซื้อสีน้ำใหม่
/fyi ลูกค้าบอกอยากได้โทนสีเย็น
/fyi deadline วันที่ 25
```

### ถ้าน่าสนใจ
```
/fyi --interesting เพิ่งเห็นเทคนิคใหม่ใน Pinterest
```

### ถ้าสำคัญมาก
```
/fyi --important ลูกค้าโทรมาเลื่อน deadline
```

---

## ดูโน้ตที่เก็บไว้

พิมพ์แค่ `/fyi` (ไม่ต้องใส่อะไร)

จะแสดงรายการโน้ตทั้งหมด

---

## ตัวอย่างสำหรับ Artist

**ระหว่างวาด:**
```
/fyi แก้ไขตาซ้ายให้เล็กลง
/fyi ลูกค้าชอบ pose ที่ 2
/fyi ต้องส่งไฟล์ขนาด 300 DPI
```

**ไอเดีย:**
```
/fyi --interesting อยากลองวาด cyberpunk
/fyi สี palette ที่ชอบ: #FF6B6B, #4ECDC4
```

**งาน:**
```
/fyi --important ลูกค้าจ่ายแล้ว 50%
/fyi ต้องส่ง WIP พรุ่งนี้
```
