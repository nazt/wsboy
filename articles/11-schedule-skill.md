# /schedule — ดูปฏิทิน

## มันคืออะไร?

`/schedule` ช่วยดู**นัดหมาย**และ**กำหนดการ**

ง่ายกว่าเปิดปฏิทินบนมือถือ

---

## ใช้ยังไง?

### ดูสัปดาห์หน้า
```
/schedule
```

### ดูวันนี้
```
/schedule today
```

### ดูเดือนที่ต้องการ
```
/schedule january
```

### ค้นหานัด
```
/schedule client
```

---

## ได้อะไรออกมา?

```
## Schedule

| วัน | เวลา | นัด |
|-----|------|-----|
| จ. 20 | 14:00 | Call ลูกค้า |
| อ. 21 | 10:00 | ส่งงาน draft |
| พ. 22 | - | ว่าง |
| พฤ. 23 | 16:00 | Meeting team |
```

---

## ต่างจาก /standup ยังไง?

| /standup | /schedule |
|----------|-----------|
| **ภาพรวมวัน** | **ปฏิทินโดยเฉพาะ** |
| มีหลายส่วน | เน้นนัดหมายอย่างเดียว |
| ใช้ตอนเช้า | ใช้ตอนไหนก็ได้ |

ลองนึกภาพว่าคุณถาม Claude ว่า:

---

## ตัวอย่างการใช้

### อยากรู้ว่าวันไหนว่าง
```
/schedule
```
→ ดูว่ามีนัดวันไหน วันไหนว่าง

### อยากหานัดกับคนนี้
```
/schedule client-name
```
→ ค้นหานัดที่มีคำนี้

### วางแผนเดือนหน้า
```
/schedule february
```
→ ดูนัดเดือนกุมภาพันธ์

---

## ข้อจำกัด

- ต้องมีไฟล์ `schedule.md` อยู่ก่อน
- Claude อ่านจากไฟล์นี้ ไม่ได้ sync กับ Google Calendar โดยตรง

---

## Tips

1. update ปฏิทินในไฟล์ `schedule.md` เป็นประจำ
2. ใช้ `/schedule today` ดูนัดวันนี้
3. ใช้ร่วมกับ `/standup` ตอนเช้า
