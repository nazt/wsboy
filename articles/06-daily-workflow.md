# Workflow ประจำวันสำหรับ Artist

## เช้า — เริ่มวัน

### 1. เช็คสถานะ
```
/standup
```
Claude จะบอก:
- งานค้าง
- นัดวันนี้
- ทำอะไรไปเมื่อวาน

### 2. หรือถ้าเพิ่งเปิด session ใหม่
```
/recap
```
สรุปว่าครั้งก่อนทำอะไรค้างไว้

---

## ระหว่างวัน — ทำงาน

### บันทึกอารมณ์
```
/feel focused energy:4
/feel tired trigger:ลงสีนาน
```

### จดโน้ตเร็ว
```
/fyi ลูกค้าบอกให้แก้ตรงนี้
/fyi --interesting เห็นเทคนิคใหม่
```

### เรียนจากวิดีโอ
```
/watch [ลิงก์ YouTube]
```

---

## ตอนเย็น — จบวัน

### สรุปงาน
```
/rrr
```
Claude เขียนสรุป + บทเรียนให้

### เตรียมส่งต่อ
```
/forward
```
จดว่าพรุ่งนี้ต้องทำอะไรต่อ

---

## สรุป Flow

```
เช้า:     /standup หรือ /recap
          ↓
ระหว่างวัน: /feel, /fyi, /watch
          ↓
เย็น:     /rrr แล้ว /forward
```

---

## ไม่ต้องใช้ทุกอัน!

เลือกใช้แค่ที่ชอบ:

| ถ้าคุณ... | ใช้อันนี้ |
|----------|----------|
| ลืมง่าย | /fyi |
| อยากดูแลตัวเอง | /feel |
| อยากเห็นความก้าวหน้า | /rrr |
| ชอบดู tutorial | /watch |
