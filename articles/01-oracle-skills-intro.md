# Oracle Skills คืออะไร? (สำหรับคนที่ไม่ใช่โปรแกรมเมอร์)

## เปรียบเทียบง่ายๆ

ลองนึกภาพว่า Claude (AI ที่คุณคุยอยู่) เหมือน**ผู้ช่วยส่วนตัว**

ปกติผู้ช่วยก็ช่วยได้ทั่วไป แต่ถ้าเราสอนเขาเพิ่มล่ะ?

- สอนให้จดบันทึกอารมณ์
- สอนให้สรุปงานตอนจบวัน
- สอนให้จำสิ่งสำคัญ
- สอนให้ดูวิดีโอ YouTube แล้วสรุปให้

**Oracle Skills** ก็คือ "ทักษะพิเศษ" ที่เราเพิ่งติดตั้งให้ Claude นั่นเอง!

---

## ติดตั้งไปแล้วมีอะไรบ้าง?

| Skill | ทำอะไร | ใช้ยังไง |
|-------|--------|----------|
| `/feel` | บันทึกอารมณ์ | พิมพ์ `/feel เหนื่อย` |
| `/rrr` | สรุปงานที่ทำ | พิมพ์ `/rrr` ตอนจบงาน |
| `/forward` | ส่งต่อให้ session หน้า | พิมพ์ `/forward` ก่อนปิด |
| `/fyi` | จดโน้ตสั้นๆ | พิมพ์ `/fyi ต้องซื้อสีใหม่` |
| `/recap` | ดูว่าเมื่อกี้ทำอะไร | พิมพ์ `/recap` |
| `/standup` | เช็คงานประจำวัน | พิมพ์ `/standup` ตอนเช้า |
| `/watch` | เรียนจาก YouTube | พิมพ์ `/watch [ลิงก์]` |

---

## หลักคิดของ Oracle

> "AI เป็นเหมือนสมองภายนอก ไม่ใช่หัวหน้า"

Oracle ออกแบบมาให้ AI **ช่วยจำ** และ **ช่วยจัด** — ไม่ใช่มาสั่ง

เหมาะกับคนที่:
- ชอบทำงานหลายอย่างพร้อมกัน
- ลืมง่าย
- อยากมี "สมองสำรอง" ช่วยจดโน้ต
