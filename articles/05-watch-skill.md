# /watch — เรียนจาก YouTube

## มันคืออะไร?

`/watch` ให้คุณ**ส่งลิงก์ YouTube** แล้ว Claude จะ:
1. ดูวิดีโอ (ผ่าน Gemini)
2. สรุปให้
3. บันทึกเป็นความรู้

---

## ใช้ยังไง?

```
/watch https://youtube.com/watch?v=xxxxx
```

แค่นี้!

---

## ตัวอย่างสำหรับ Artist

**Tutorial การวาด:**
```
/watch https://youtube.com/watch?v=xxxxx
```
→ ได้สรุป: "วิดีโอนี้สอนการวาดตา 3 แบบ..."

**Art Tips:**
```
/watch https://youtu.be/yyyyy
```
→ ได้: "5 เทคนิคที่สอน: 1) ใช้ reference 2) เริ่มจาก shapes..."

---

## ทำไมถึงมีประโยชน์?

1. **ประหยัดเวลา** — วิดีโอ 1 ชม. อ่านสรุป 5 นาที
2. **ค้นหาได้** — ถาม Claude ทีหลังว่า "เทคนิคไหนที่เรียนมา?"
3. **ไม่ต้องจด** — Claude จดให้แล้ว

---

## ข้อจำกัด

- ต้องมี Gemini (Google AI) ทำงานอยู่
- วิดีโอยาวอาจใช้เวลานาน
- บางวิดีโอที่ถูกบล็อกอาจดูไม่ได้

## Workflow สำหรับเรียนวาด

1. หาวิดีโอที่อยากดูใน YouTube
2. `/watch [ลิงก์]`
3. อ่านสรุป
4. ลองทำตาม
5. `/fyi เทคนิคที่ชอบจากวิดีโอนี้คือ...`
